Input Attributes

Attribute Data Type Description
Weather Historical string Polygon of area of interest is entered in format [[longitude1,latitude1],[longitude2,latitude2],...,[longitude1,latitude1]]Polygon string should be circular and used in EPSG:4326 format.
start_date string Solar radiation (in watt) provides both light and heat energy that is necessary for living organisms
end_date string date in format YYYY-MM-DD for which historical weather data is shown
duration string Daily light integral (in moles) used to evaluate amount of light received by plants

Response Parameters

Attribute Description
Weather Historical Historical weather data of specified date range
SolarRadiation_watt Solar radiation (in watt) provides both light and heat energy that is necessary for living organisms
date date in format YYYY-MM-DD for which historical weather data is shown
DailyLightIntegral_moles Daily light integral (in moles) used to evaluate amount of light received by plants
Humidity_Percent Humidity (in percentage) gives amount of water vapour present in air and holds importance in surface life
MinHumidity_Percent Minimum Humidity (in percentage) gives minimum amount of water vapour in air
MaxHumidity_Percent Maximum Humidity (in percentage) gives maximum amount of water vapour in air
